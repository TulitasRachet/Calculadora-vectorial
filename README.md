# 📊 Calculadora Vectorial
Calculadora de vectores con visualización en **2D** y **3D**.

---

## 📝 Descripción
Este proyecto es una aplicación web que permite realizar operaciones vectoriales entre hasta tres vectores (**A, B y C**) y visualizar los resultados en **2D** o en **3D**.  

Las operaciones disponibles incluyen:
- ➕ **Suma de vectores**
- ➖ **Resta de vectores**
- 📐 **Cálculo del ángulo entre vectores**
- ⚪ **Producto punto**
- ✖ **Producto cruz**
- 🔺 **Producto triple escalar** `a · (b × c)`
- `a × (b × c)`
- `(a × b) × c`

Además, se puede elegir cómo visualizar los vectores:  
📉 **Plano 2D** o 📊 **Espacio 3D**.

🔗 **Versión en línea:**  
👉 [Calculadora Vectorial](https://tulitasrachet.github.io/Calculadora-vectorial/)

## 🚀 ¿Cómo ejecutar?

La calculadora está disponible en línea en:  
👉 [https://tulitasrachet.github.io/Calculadora-vectorial/](https://tulitasrachet.github.io/Calculadora-vectorial/)  

Si quieres ejecutarla de forma local, tienes dos opciones:

### 🔹 Descargar el archivo
1. Entra al repositorio: [Calculadora Vectorial](https://github.com/TulitasRachet/Calculadora-vectorial).  
2. Descarga el archivo `index.html`.  
3. Ábrelo en tu navegador.  

### 🔹 Clonar el repositorio
1. Asegúrate de tener **Git** instalado.  
2. Abre tu terminal y ejecuta:
